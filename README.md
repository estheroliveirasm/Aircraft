Aircraft - Fleet and Air Logistics Manager

Developed for the Programming Language 1 course, this system manages aircraft and routes using linked lists for memory optimization. It supports both cargo and passenger aircraft, tracking availability, maintenance, and operational limits.

Data persistence is handled via binary files for efficient storage of complex structures. In addition to a terminal interface, the project exports reports in TXT, CSV, and HTML formats. The system also processes flight statistics, such as fuel consumption and fleet utilization.

Developers: Beatriz Goulart and Esther de Oliveira

Institution: UFSC - Federal University of Santa Catarina

Rating: 10/10 – Full implementation of all specifications and robust input validation.
